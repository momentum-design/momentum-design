## Accessibility

The `mdc-list` component is designed to be used in combination with other
components like `mdc-listheader` and `mdc-listitem` to create accessible lists.

### Checklist

- Ensure to pass in a `aria-label` or `aria-labelledby` to indicate the purpose
  of the list.
- Make sure to use `mdc-listitem` components for each item in the list to ensure
  proper semantics.
- Use `mdc-listheader` for headers within lists to provide context (if
  applicable).
