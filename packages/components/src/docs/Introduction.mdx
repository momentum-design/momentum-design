> ⚠️ **Note:**  
> Momentum Design does **not support Internet Explorer 11** or **pre-Chromium Edge**.  
> While web components can be partially polyfilled for legacy browsers, this is **outside the scope** of this project.  
> If you’re using Momentum Design in such browsers, you’re gonna have a bad time. ⛷
