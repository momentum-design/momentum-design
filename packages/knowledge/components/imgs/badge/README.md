# Badge guideline images (`imgs/badge`)

PNG exports of Figma frames named **IMG** from [Momentum Component Guidelines — Badge](https://www.figma.com/design/MrrkzCQ0YXzUTca2XhesDZ/%F0%9F%93%9C-Momentum-Component-Guidelines?node-id=814-19379).

## Generate

```bash
export FIGMA_ACCESS_TOKEN="<figma-personal-access-token>"
yarn workspace @momentum-design/knowledge export:badge-guidelines-imgs
```

Writes **8×** PNGs into this folder. See [Figma API access tokens](https://www.figma.com/developers/api#access-tokens).
