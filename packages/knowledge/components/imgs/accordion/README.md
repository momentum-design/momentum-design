# Accordion guideline images (`imgs/accordion`)

PNG exports of Figma frames named **IMG** from [Momentum Component Guidelines — Accordion](https://www.figma.com/design/MrrkzCQ0YXzUTca2XhesDZ/%F0%9F%93%9C-Momentum-Component-Guidelines?node-id=10257-1136161).

## Generate

```bash
export FIGMA_ACCESS_TOKEN="<figma-personal-access-token>"
yarn workspace @momentum-design/knowledge export:accordion-guidelines-imgs
```

Writes **2×** PNGs into this folder. See [Figma API access tokens](https://www.figma.com/developers/api#access-tokens).
