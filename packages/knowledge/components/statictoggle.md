# StaticToggle (`mdc-statictoggle`)

## Development

### Summary

This is a decorative component that is styled to look as a toggle. 

It has 5 properties - checked, size, disabled, readonly and soft-disabled. 

We are using the same styling that has been created for the `mdc-toggle` component.

### Source

- Component folder: [`packages/components/src/components/statictoggle/`](../../components/src/components/statictoggle/)
- Built metadata references: `components/statictoggle/statictoggle.component.js` (from Custom Elements Manifest)

### Install and global setup

Install the library:

```bash
npm install @momentum-design/components
```

Load fonts and token CSS, set typography class, and use **ThemeProvider** / **IconProvider** where needed. Follow the full checklist in [Consumption.mdx](../../components/src/docs/Consumption.mdx) (imports, HTML example, webpack/TS notes).

### Import this component

**Web component** (registers the custom element):

```javascript
import '@momentum-design/components/components/statictoggle';
```

```html
<mdc-statictoggle></mdc-statictoggle>
```

**React** wrapper (from `@lit/react` codegen):

```javascript
import { StaticToggle } from '@momentum-design/components/react';
```

```jsx
<StaticToggle />
```
